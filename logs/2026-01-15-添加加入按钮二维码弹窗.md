# 2026-01-15 - 添加加入按钮二维码弹窗

## 用户需求
用户希望在点击"加入"按钮时，弹出范西瓜.jpg二维码图片。

## 执行的操作
1. 在index.html中添加了二维码弹窗模态框（QR Code Modal）
2. 为"加入"按钮添加了id="joinButton"
3. 创建了QR Code Modal模块，实现：
   - 点击"加入"按钮打开弹窗
   - 点击关闭按钮或背景关闭弹窗
   - 按ESC键关闭弹窗
   - 显示范西瓜.jpg二维码图片

## 修改的文件
- index.html: 添加了二维码弹窗HTML结构和JavaScript功能

## 实现方式
- 复用了现有的modal样式系统
- 使用简洁的JavaScript实现弹窗的打开和关闭功能
- 支持键盘ESC键和点击背景关闭弹窗
